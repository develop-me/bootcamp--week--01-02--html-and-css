## Object Oriented CSS

---

### Separates structure from skin, style from layout

Encourages reuseable, scalable CSS

Example: button used on site, regardless of code surrounding it

---

```css
.button {
    background-color: green;
    border: 1px solid blue;
}

form .button {
    width: 100px;
    margin: 0px auto;
}

.card .button {
    display: block;
    width: 200px;
}
```
