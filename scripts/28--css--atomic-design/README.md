# Atomic Design

---

### Way of thinking about structure:

- **Atoms:** Elements (input, p, button etc...)
- **Molecules:** Set of elements (search form)
- **Organism:** Set of Molecules (site header)
- **Templates:** Set of Organisms (repeated)
- **Pages:** Final site
